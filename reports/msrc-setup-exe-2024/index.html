MSRC: Setup.exe Security Bypass (CVE-2024-38202 / CVE-2024-21302)

Thu, May 1, 2025

Analysis of Windows 11 setup stack vulnerabilities that allow bypassing hardware enforcement, TPM checks, and VBS policy via LabConfig registry manipulation.

  • Summary
  • Impact
  • Read More

Summary

Two chained vulnerabilities in the Windows 11 installation stack allow an attacker with local access to bypass hardware enforcement checks enforced by setup.exe and winload.efi. By manipulating LabConfig registry keys, the setup process silently skips TPM 2.0, Secure Boot, and CPU generation requirements — and more critically, can silently downgrade VBS/HVCI policy protections.

Impact

  • Silently disables Virtualization Based Security on upgraded systems
  • Bypasses TPM 2.0 and Secure Boot requirements without user notification
  • Allows installing a non-compliant Windows 11 build that reports hardware compliance to management tools
