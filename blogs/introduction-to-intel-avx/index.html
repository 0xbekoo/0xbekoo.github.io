Introduction to AVX/AVX2

In 2013, Intel was introducted ‘AVX2’ for Haswell microarchitecture. AVX itself is actually SIMD extension to the x86 instruction set architecture for microprocessors. The main purpose of this extension is that it provides new features, new instructions, and a new coding scheme.
